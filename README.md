# react-native-wemap-livemap

Wemap Livemap

## Installation

```sh
npm install react-native-wemap-livemap
```

## Usage

```js
import Livemap from 'react-native-wemap-livemap';

// ...

const MyMap = () => (
  <Livemap mapId={-1} token={'7ETI43N4ZZGARWPHJ57WQAARW'} style={{ flex: 1 }} />
);
```
